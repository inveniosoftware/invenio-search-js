Invenio-Search client library for building the web UI.

Installation
------------

    $ npm i

Demo
----

    $ pushd examples/simple; npm install; popd
    $ npm run-script demo

Navigate to `http://localhost:8000` to see the `demo`.

Tests
-----

    $ npm test

Docs
----

    $ # build
    $ npm run-script docs
    $ # read
    $ open docs/index.html
