Invenio-Search client library for building the web UI.

Installation
------------

.. code-block:: console

    $ npm i

Demo
----
.. code-block:: console

    $ cd example; npm install; cd ..
    $ gulp demo

Navigate to ``http://localhost:8000`` to see the ``demo``

Tests
-----
.. code-block:: console

    $ gulp test

Docs
----

.. code-block:: console

    $ # build
    $ gulp docs
    $ # read
    $ open docs/index.html

How to use?
-----------

Check out the ``example/`` to see how to configure your app
